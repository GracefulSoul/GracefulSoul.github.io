    к°қмІҙм§Җн–Ҙ м„Өкі„(SOLID)

    • лЎңлІ„нҠё л§ҲнӢҙмқҙ 2000л…„лҢҖ мҙҲл°ҳм—җ лӘ…лӘ…н•ң к°қмІҙ м§Җн–Ҙ н”„лЎңк·ёлһҳл°Қ1 л°Ҹ м„Өкі„мқҳ лӢӨм„Ҝ к°Җм§Җ кё°ліё мӣҗм№ҷмқ„ л§ҲмқҙнҒҙ нҺҳлҚ”мҠӨк°Җ л‘җл¬ёмһҗм–ҙ кё°м–өмҲ лЎң мҶҢк°ңн•ң кІғмқҙлӢӨ.
    • н”„лЎңк·ёлһҳлЁёк°Җ мӢңк°„мқҙ м§ҖлӮҳлҸ„ мң м§Җ ліҙмҲҳмҷҖ нҷ•мһҘмқҙ мү¬мҡҙ мӢңмҠӨн…ңмқ„ л§Ңл“Өкі мһҗ н•  л•Ң мқҙ мӣҗм№ҷл“Өмқ„ н•Ёк»ҳ м Ғмҡ©н•  мҲҳ мһҲлӢӨ.
    • SOLID мӣҗм№ҷл“ӨмқҖ мҶҢн”„нҠёмӣЁм–ҙ мһ‘м—…м—җм„ң н”„лЎңк·ёлһҳлЁёк°Җ мҶҢмҠӨ мҪ”л“ңк°Җ мқҪкё° мүҪкі  нҷ•мһҘн•ҳкё° мүҪкІҢ лҗ  л•Ңк№Ңм§Җ мҶҢн”„нҠёмӣЁм–ҙ мҶҢмҠӨ мҪ”л“ңлҘј лҰ¬нҢ©н„°л§Ғн•ҳм—¬ мҪ”л“ң лғ„мғҲлҘј м ңкұ°н•ҳкё° мң„н•ҙ м Ғмҡ©н•  мҲҳ мһҲлҠ” м§Җм№ЁмқҙлӢӨ.
    • мқҙ мӣҗм№ҷл“ӨмқҖ м• мһҗмқј мҶҢн”„нҠёмӣЁм–ҙ к°ңл°ңкіј м Ғмқ‘м Ғ мҶҢн”„нҠёмӣЁм–ҙ к°ңл°ңмқҳ м „л°ҳм Ғ м „лһөмқҳ мқјл¶ҖлӢӨ.

    лӢЁмқј мұ…мһ„мқҳ мӣҗм№ҷ(Single Responsibility Principle - SRP)

    • н•ң нҒҙлһҳмҠӨлҠ” н•ҳлӮҳмқҳ мұ…мһ„л§Ң к°Җм ём•ј н•ңлӢӨ.
    public class Music {

  private String name;
  private String artist;
  private LocalDateTime releaseDate;

  public String getName() {
    return name;
  }

  public void setName(String name) {
    this.name = name;
  }

    • к·ёлһҳм„ң м•„лһҳмҷҖ к°ҷмқҙ мқҢм•… мһ¬мғқм—җ лҢҖн•ң нҒҙлһҳмҠӨлҘј л¶„лҰ¬н•ҙм•ј н•ҳмҳҖлӢӨ.

    лҰ¬мҠӨмҪ”н”„ м№ҳнҷҳ мӣҗм№ҷ(Liskov Substitution Principle - LSP)

    • н”„лЎңк·ёлһЁмқҳ к°қмІҙлҠ” н”„лЎңк·ёлһЁмқҳ м •нҷ•м„ұмқ„ к№ЁлңЁлҰ¬м§Җ м•Ҡмңјл©ҙм„ң н•ҳмң„ нғҖмһ…мқҳ мқёмҠӨн„ҙмҠӨлЎң л°”кҝҖ мҲҳ мһҲм–ҙм•ј н•ңлӢӨ.
    public class Item {

  private int price;

  public int getPrice() {
    return price;
  }

  public void setPrice(int price) {
    this.price = price;
  }

}
public class DiscountItem extends Item {

  private double discountRate;

  public double getDiscountRate() {
    return discountRate;
  }

  public void setDiscountRate(double discountRate) {
    this.discountRate = discountRate;
    this.applyDiscountedPrice();
  }

  private void applyDiscountedPrice() {
    super.setPrice((int)(super.getPrice() * (1 - this.discountRate)));
  }

}
public class WrongDiscountItem extends Item {

  private double discountRate;

  public double getDiscountRate() {
    return discountRate;
  }

  public void setDiscountRate(double discountRate) {
    this.discountRate = discountRate;
  }

  @Override
  public void setPrice(int price) {
    super.setPrice((int)(super.getPrice() * (1 - this.discountRate)));
  }

}
    • мң„мқҳ мҪ”л“ңм—җм„ң Itemкіј WrongDiscountItemмқҖ setPrice л©”м„ңл“ңлҘј мҲҳн–ү мӢң кІ°кіј к°’мқҙ лӢ¬лқјм§ҖлҜҖлЎң LSPлҘј мң„л°ҳн•ҳл©°, Itemкіј DiscountItemмқҖ LSPлҘј мӨҖмҲҳн•ңлӢӨ.

    public class WrongComputer {
