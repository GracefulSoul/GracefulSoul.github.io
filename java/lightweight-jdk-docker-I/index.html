    к°ңмҡ”

    • Java м• н”ҢлҰ¬мјҖмқҙм…ҳмқҖ JVM(Java Virtual Machine)мқҙ н•Ёк»ҳ м»Ён…Ңмқҙл„Ҳм—җ л°°нҸ¬лҗҳм–ҙм•ј н•ҳкё° л•Ңл¬ём—җ л№„көҗм Ғ нғҖ м–ём–ҙлЎң л°°нҸ¬лҗҳлҠ” м»Ён…Ңмқҙл„ҲліҙлӢӨ мҡ©лҹүмқҙ л§Өмҡ° нҒ¬лҜҖлЎң, мқҙ л¶Җл¶„мқ„ н•ҙмҶҢн•ҳкё° мң„н•ң кІҪлҹүнҷ”лҗң Dockerizng1мқ„ Amazon Crretto JDK2лҘј мқҙмҡ©н•ҳм—¬ м„ӨлӘ…н•ҳкі мһҗн•ңлӢӨ.
    • м—¬кё°м„ңлҠ” Custom JREлҘј мғқм„ұн•ҳлҠ” Dockerмқҳ Multi-Stage builds3лҘј нҷңмҡ©н•ҳм—¬ мӢӨм ң л№Ңл“ңн•ҳлҠ” JAVA м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ кІҪлҹүнҷ”н•ңлӢӨ.

    Stage 1

    • Stage 1м—җм„ңлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳ мқҳмЎҙ кҙҖкі„лҘј л¶„м„қн•ҳм—¬ кІҪлҹүнҷ”лҗң Custom JREлҘј л§Ңл“ңлҠ” л¶Җл¶„м—җ мӨ‘м җмқ„ л‘”лӢӨ.
    • л¶Җк°Җм ҒмңјлЎң л№Ңл“ңлҗң jar нҢҢмқјмқ„ н•„мҡ”н•ң нҢҢмқјл“Өл§Ң мӮ¬мҡ©н•ҳкё° мң„н•ҳм—¬ м••м¶• н•ҙм ңн•ңлӢӨ.
    # Stage 1. Create custom JRE
FROM amazoncorretto:21-alpine AS jrebuilder

# Add the application jar to the container
COPY ./build/libs/hello-docker-*-SNAPSHOT.jar /app.jar

# Install binutils
RUN apk add --no-cache binutils

# Extract jar file and generate custom JRE using dependency
RUN mkdir -p /app && (cd /app; jar -xf /app.jar) \
	&& DEPENDENCY=$(jdeps --ignore-missing-deps --print-module-deps --recursive --multi-release 21 --class-path="/app/BOOT-INF/lib/*" /app.jar) \
	&& ${JAVA_HOME}/bin/jlink \
		--verbose \
		--add-modules ${DEPENDENCY} \
		--strip-debug \
		--no-man-pages \
		--no-header-files \
		--compress=2 \
		--output \
		customjre

    Jdeps5

    • JdepsлҠ” JAVA Classмқҳ мқҳмЎҙм„ұ л¶„м„қмқ„ мң„н•ң лҸ„кө¬лЎң, лӘЁл“Ҳнҷ”к°Җ м Ғмҡ©лҗң JAVA 9 лІ„м „ мқҙмғҒл¶Җн„° мӮ¬мҡ©мқҙ к°ҖлҠҘн•ҳлӢӨ.
    Option Description
    вҖ“ignore-missing-deps мҷёмЎҙм„ұмқҙ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” лӘЁл“ҲмқҖ м ңмҷё.
    вҖ“print-module-deps Jlinkм—җм„ң мҡ”кө¬н•ҳлҠ” нҸ¬л©§м—җ л§һлҠ” мүјн‘ңлЎң кө¬л¶„лҗң лӘЁл“Ҳ мқҳмЎҙм„ұ лӘ©лЎқмқ„ м¶ңл Ҙ.
    вҖ“recursive лӘЁл“  лҹ°нғҖмһ„ мў…мҶҚм„ұмқ„ мһ¬к·Җм ҒмңјлЎң нғҗмғү.
    вҖ“multi-release ${VERSION} мқҳмЎҙм„ұмқ„ л¶„м„қн•  JAVA лІ„м „мқ„ ${VERSION}м—җ лӘ…мӢң. лӢЁ, лӘЁл“Ҳнҷ”к°Җ м Ғмҡ©лҗң 9 лІ„м „ мқҙмғҒл§Ң м Ғмҡ©.
    вҖ“class-path=вҖқ${PATH}вҖқ мқҳмЎҙм„ұмқ„ л¶„м„қн•  class нҢҢмқјл“Өмқ„ м°ҫмқ„ мң„м№ҳлҘј ${PATH}м—җ м§Җм •.

    Jlink6

    • JlinkлҠ” JdepsлҘј мқҙмҡ©н•ҳм—¬ л¶„м„қлҗң мқҳмЎҙм„ұмқ„ нҷңмҡ©н•ҳм—¬ мөңм Ғнҷ”лҗң Custom JREлҘј кө¬м„ұн•  мҲҳ мһҲлҠ” лҸ„кө¬мқҙлӢӨ.
    Option Description
    вҖ“verbose м§„н–ү мӨ‘ мғҒм„ё лӮҙм—ӯмқ„ м¶ңл Ҙ.
    вҖ“add-modules ${DEPENDENCY} jdepsлҘј мқҙмҡ©н•ҳм—¬ л¶„м„қн•ң мүјн‘ңлЎң кө¬л¶„лҗң лӘЁл“Ҳ мқҳмЎҙм„ұ лӘ©лЎқ
    вҖ“strip-debug л””лІ„к·ё м •ліҙлҘј м ңкұ°.
    вҖ“no-man-pages man page(нҠ№м • лӘ…л №мқҙлӮҳ мһҗмӣҗл“Өмқҳ л©”лүҙм–јмқ„ м¶ңл Ҙн•ҳлҠ” мҳҒм—ӯ) лҜёмӮ¬мҡ©.
    вҖ“no-header-files header files(лЎңм§Ғмқҙ м ҖмһҘлҗң нҢҢмқј) лҜёмӮ¬мҡ©.
    вҖ“compress=2 лҰ¬мҶҢмҠӨ м••м¶• м—¬л¶Җ. 0 : лҜёмӮ¬мҡ©, 1 : мғҒмҲҳ л¬ёмһҗм—ҙ кіөмң , 2 : м••м¶•мқ„ мқҳлҜён•ҳл©°, ліҙнҺём ҒмңјлЎң 2лҘј мӮ¬мҡ©.
    вҖ“output=${PATH} лҰ¬мҶҢмҠӨлҘј м ҖмһҘн•  мң„м№ҳлҘј ${PATH}м—җ м§Җм •.

    Stage 2

    • Stage 2м—җм„ңлҠ” Stage 1м—җм„ң л¶„м„қлҗң мқҳмЎҙм„ұмқ„ нҶөн•ҙ мҷ„м„ұлҗң Custom JREмҷҖ м••м¶• н•ҙм ңн•ң jarм—җм„ң н•„мҡ”н•ң нҢҢмқјлЎң мқҙлҜём§ҖлҘј кө¬м„ұн•ҳм—¬ кІҪлҹүнҷ”лҗң JAVA м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…Ңмқҙл„ҲлҘј кө¬м„ұн•ңлӢӨ.
    # Stage 2. Make container for application
FROM alpine:3.20
ENV JAVA_HOME=/jre
ENV PATH="${JAVA_HOME}/bin:${PATH}"
ARG DEPENDENCY=/app

# Add Maintainer Info
LABEL maintainer="GracefulSoul on <gracefulsoul@github.com>"

# Copy custom JRE
COPY --from=jrebuilder /customjre ${JAVA_HOME}

# Copy extract files in jar
COPY --from=jrebuilder ${DEPENDENCY}/BOOT-INF/lib ${DEPENDENCY}/lib
COPY --from=jrebuilder ${DEPENDENCY}/META-INF ${DEPENDENCY}/META-INF
COPY --from=jrebuilder ${DEPENDENCY}/BOOT-INF/classes ${DEPENDENCY}

# Move work directory
WORKDIR ${DEPENDENCY}

# Run application
ENTRYPOINT [ "java", "-cp", "${DEPENDENCY}:${DEPENDENCY}/lib/*", "gracefulsoul.HelloDockerApplication" ]

    • Stage 1м—җм„ң Custom JREлҘј л§Ңл“Өм—ҲмңјлҜҖлЎң, кІҪлҹүнҷ”лҗң Alpine Linux7 мқҙлҜём§ҖлҘј м•„лһҳмқҳ нҷҳкІҪ м„Өм •мқ„ мқҙмҡ©н•ҳм—¬ мӮ¬мҡ©н•ңлӢӨ.
    • JAVA_HOME нҷҳкІҪ ліҖмҲҳлҘј м¶”к°Җн•ҳкё° мң„н•ҳм—¬ JAVA_HOMEмқҖ вҖң/jreвҖқлЎң мӨҖ нӣ„ PATHм—җ вҖңJAVA_HOME/binвҖқмқ„ м¶”к°Җн•ңлӢӨ.
    • вҖң/appвҖқ нҸҙлҚ”лҘј м „м—ӯ ліҖмҲҳлЎң мӮ¬мҡ©н•ҳкё° мң„н•ң DEPENDENCYлҘј м •мқҳн•ңлӢӨ.

    Copy custom JRE

    • Stage 1м—җм„ң мғқм„ұн•ң Custom JREк°Җ м ҖмһҘлҗң вҖң/customjreвҖқ нҸҙлҚ”лҘј вҖң/jreвҖқлЎң ліөмӮ¬н•ңлӢӨ.

    Copy extract files in jar

    • jar нҢҢмқјм—җм„ң JAVA м• н”ҢлҰ¬мјҖмқҙм…ҳ кө¬лҸҷмқ„ мң„н•ҙ н•„мҡ”н•ң м•„лһҳмқҳ нҸҙлҚ”л“Өмқ„ ліөмӮ¬н•ңлӢӨ.
    • мқҳмЎҙм„ұ мЈјмһ…лҗң Libraryк°Җ м ҖмһҘлҗң вҖң/app/BOOT-INF/libвҖқ нҸҙлҚ”лҘј вҖң/app/libвҖқ нҸҙлҚ”лЎң ліөмӮ¬н•ңлӢӨ.
    • л©”нғҖ лҚ°мқҙн„°мҷҖ м„Өм • м •ліҙлҘј м ҖмһҘлҗң вҖң/app/META-INFвҖқ нҸҙлҚ”лҘј вҖң/app/META-INFвҖқ нҸҙлҚ”лЎң ліөмӮ¬н•ңлӢӨ.
    • м»ҙнҢҢмқјлҗң class нҢҢмқјмқҙ м ҖмһҘлҗң вҖң/app/BOOT-INF/classesвҖқ нҸҙлҚ”лҘј вҖң/appвҖқ нҸҙлҚ”лЎң ліөмӮ¬н•ңлӢӨ.

    • ENTRYPOINTмқё м»Ён…Ңмқҙл„Ҳк°Җ мӢӨн–үлҗ  л•Ң кё°ліёмңјлЎң лҸҷмһ‘н•ҳлҠ” лӘ…л №м–ҙлҠ” м•„лһҳмқҳ мЎ°н•©мңјлЎң мҷ„м„ұн•ңлӢӨ.
    • вҖңjavaвҖқ лӘ…л №м–ҙлҠ” JAVA м• н”ҢлҰ¬мјҖмқҙм…ҳ мӢӨн–үмқ„ мң„н•ң лӘ…л №м–ҙмқҙлӢӨ.
    • вҖң-cp /app:/app/lib/вҖңмқҖ classpathлҘј м§Җм •н•ҳлҠ” лӘ…л №м–ҙлЎң, м»ҙнҢҢмқјлҗң class нҢҢмқј лЈЁнҠё мң„м№ҳмқё вҖң/appвҖқкіј мқҳмЎҙм„ұмңјлЎң м¶”к°Җлҗң Libraryл“Өмқ„ нҸ¬н•Ён•ҳлҠ” вҖң/app/lib/вҖңмқ„ мқҙм–ҙмӨҖлӢӨ.
